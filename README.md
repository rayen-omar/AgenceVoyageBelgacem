# Module Agence Voyage Belgacem

Module Odoo 16 pour la gestion complète d'une agence de voyage.

## Installation

1. Placez le module dans le dossier `addons` d'Odoo
2. Mettez à jour la liste des applications
3. Installez le module "Agence Voyage Belgacem"
4. Configurez la devise TND (Dinar Tunisien) dans Paramètres > Société > Devise

## Fonctionnalités

- **Destinations** : Gestion des lieux de voyage
- **Clients** : Gestion de la base clients
- **Hébergements** : Gestion des hôtels et hébergements
- **Forfaits Touristiques** : Création et gestion des forfaits avec itinéraires
- **Réservations** : Gestion complète des réservations avec transport, vols et facturation
- **Paiements** : Suivi des paiements et reçus
- **Factures** : Création automatique de factures depuis les réservations
